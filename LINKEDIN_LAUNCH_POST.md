# LinkedIn launch post

I kept checking Claude and Codex usage limits while working, so I built a better way to keep them visible.

Introducing **Quota Deck** — a small companion app that turns an unused phone, tablet, or iPad into a live AI usage dashboard.

It shows:

• Current five-hour and weekly usage
• Allowance remaining
• Live reset countdowns
• Context-window usage and tokens left
• Claude Desktop, Claude Code, and Codex in one focused display

The important part for me was privacy: Quota Deck runs locally, pairs screens with a six-digit code, and shares only usage metrics — not prompts, conversations, cookies, or AI account credentials.

The first **Windows x64 public alpha** is ready. It includes a one-click installer, automatic startup, phone/tablet layouts, and an optional Docker deployment for the dashboard relay.
